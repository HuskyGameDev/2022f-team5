Monstrous
This game is a 2d Horde Survival bulllet heaven that is currently in development. The goal is to grow stronger as you defeat enemies to gain their strength and 
to survive as long as you can. 

CONTROLS:
wasd - movement
cursor - used to aim attack
esc - pause game

INSTALL - you can unzip the zipped folder and the executable should be there.

TEAM - Colby Dummer, Jack Snowden, Wyatt Helzer, Chase Bootz, Joshua Radford, Kelso Edwards, Waylon Alvesteffer, Jacob Brendemuehl, Kirsten Love.
